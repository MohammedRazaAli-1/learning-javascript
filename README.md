
# JavaScript Learning Journey (60 Days)

This repository contains my JavaScript learning and practice work.  
It covers JavaScript from basic concepts to advanced topics, with step-by-step programs and practice problems.

This repository represents my **60-day JavaScript learning challenge**, which I completed with consistency and discipline.  
During these 60 days, I studied JavaScript **every day for around 5–6 hours**.

I divided this 2-month learning journey into **three parts**, and the folder structure of this repository follows the same order:

1. Basic JavaScript  
2. Intermediate JavaScript  
3. Advanced JavaScript  

---

## Basic JavaScript

I started my JavaScript learning from the very beginning with a simple *Hello World* program.

After that, I learned:
- Comments in JavaScript  
- Variables and types of variables  
- All data types in JavaScript  
- Operators:
  - Arithmetic  
  - Assignment  
  - Comparison  
  - Increment and decrement  
  - Logical  
  - Ternary  
- Conditional statements:
  - `if`
  - `else`
  - `else if`
- Loops:
  - `for`
  - `while`
  - `do while`
  - `forEach`
- Functions:
  - Normal functions  
  - Function expressions  
  - Arrow functions  
  - Callback functions  
  - IIFE functions  

This phase helped me understand the basic structure and flow of JavaScript programs.

---

## Intermediate JavaScript

After completing the basics, I moved to intermediate JavaScript.

In this part, I worked with:
- Arrays and their common operations and methods  
- Objects and object-based operations  

I also solved a good object-based problem in the file  
`03_object_problem.js`.

After that, I learned how the `this` keyword works in different situations.

I studied both the old and new ways of working with objects in JavaScript.

Then I started learning **Object-Oriented Programming (OOP)** in JavaScript and implemented the four main concepts:
- Encapsulation  
- Abstraction  
- Inheritance  
- Polymorphism  

I also learned about prototypes and prototype chaining.

---

## Advanced JavaScript

After completing the core JavaScript concepts, I moved to advanced JavaScript topics.

I started with **BOM (Browser Object Model)** to understand how the browser works with JavaScript.

After BOM, I learned **DOM (Document Object Model)**.  
In DOM, I studied:
- DOM tree structure  
- Nodes in the DOM tree  
- Selecting elements  
- Creating elements  
- Adding classes and IDs  
- Updating content  
- Adding styles using JavaScript  

After learning DOM, I created a **Notes Manager** project, which is a simple and functional application.

Next, I learned **events in JavaScript**, including:
- Mouse events  
- Keyboard events  
- Form events  
- Window events  

Using DOM and events, I created a **Typing Master** application.  
This project is available in the file `07_events_problem.js`.

Finally, I learned **asynchronous JavaScript**, including:
- Callback functions  
- `setTimeout`  
- Promises  

---

I followed this learning routine for **60 days without skipping any day**.  
I studied JavaScript for **5–6 hours daily** and focused on understanding concepts through practice.

This repository contains the complete record of my JavaScript learning journey.
